Projects

I believe in Free Software, so I run several software projects which have been released under Free Software licenses.

AgenDAV

AgenDAV is a CalDAV web client built on top of free software. At the moment of starting its development (May of 2011) there were no other free CalDAV web clients, so I started this one.

It has lots of features, including:

  • Drag&Drop interface
  • Multilanguage and timezones support
  • Multiple calendars
  • Configurable calendar colours
  • Calendar sharing

It just requires a working CalDAV server (e.g. DAViCal, SabreDAV, Calendar and Contacts Server, etc)

For documentation and downloads, visit AgenDAV website and AgenDAV project page.

Consigna

Consigna is a file sharing web application created to make easier sending files to other people without using large email attachments.

Main features are:

  • Clean interface
  • Progress bar while uploading a file
  • Password protected files
  • Users can manage their own uploaded files
  • Automatic file expiration
  • Files are scanned by ClamAV antivirus

Consigna is currently being used at some public institutions and companies in Spain. Some of them are:

  • University of Seville
  • University of Cádiz
  • University of Huelva
  • Sevilla City Council
  • Public Company for Agricultural and Fisheries development (Andalusia)
  • Reina Sofia Public Hospital (Córdoba, Andalusia)
  • Other companies which asked to be kept anonymous

Documentation, code and download links at Consigna project at University of Seville software forge (temporary location, pending move to Github). Unfortunately is only available in Spanish right now.

libopensso

The University of Seville had a long Sun software background, so they wanted to use OpenSSO (formerly known as Sun Access Manager) as their Single Sign-On solution.

There was no easy way to set up SSO authentication on applications without messing with Apache buggy modules, so I wrote a PHP helper for PHP applications that makes using the SSO a trivial task.

First version of this library was really rudimentary, but I have rewritten it to make it totally OOP and it's now a Composer package.
